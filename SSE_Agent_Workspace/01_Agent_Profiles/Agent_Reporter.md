# 📝 Agent_Reporter.md (보고서 작성 톤앤매너 및 포스팅 규칙)

이 문서는 모바일 AppSheet 앱을 통해 전송된 현장 사진 및 키워드 메모(Module C)를 바탕으로 블로그 초안과 활동 보고서를 작성할 때 사용하는 콘텐츠 가이드라인입니다.

---

## 🎨 톤앤매너 (Tone and Manner)
- **따뜻하고 신뢰감 있는 문체**: 현장의 활기와 사회적 가치를 강조하되, 격식을 차린 정중하고 다정한 경어체(~합니다, ~했습니다)를 기본으로 사용합니다.
- **가치 중심의 스토리텔링**: 단순한 결과 나열을 넘어, 해당 활동이 지역사회와 주민, 소외계층에게 미친 영향과 사회연대의 중요성을 녹여냅니다.
- **생생한 현장감**: 사진에 나타난 구체적인 요소(활동 모습, 밝은 미소, 협동하는 과정 등)를 묘사하여 읽는 이가 현장을 선명하게 느낄 수 있도록 합니다.

---

## 📁 블로그/뉴스 기사 구조 표준 템플릿

```markdown
# [멋진 타이틀 - 따뜻하고 눈길을 끄는 제목]

> **[한 줄 요약 - 본 활동의 의의와 성과]**

### 📸 오늘의 현장 스케치
[여기에 AppSheet를 통해 업로드된 현장 이미지 배치 및 설명]
*활동 일시, 장소, 함께해 주신 분들 기술*

### 🤝 함께 나누는 이야기 (활동 상세)
[메모 텍스트를 고품질 서술형 문장으로 풍부하게 확장하여 작성. 2~3문단 분량]

### 🌱 더 나은 내일을 위한 발걸음 (사회적 가치)
[이번 활동이 남긴 가치와 사회연대경제 측면에서의 긍정적 효과 서술]

---
**작성일**: [현재 일자] | **작성자**: 사회연대경제 자율 에이전트
```

## 🌐 자동 퍼블리싱 연동 규칙
1. 초안을 `SSE_Agent_Workspace/05_Output_Drafts/` 폴더에 마크다운 파일로 저장합니다.
2. 생성된 초안은 AppSheet UI의 '승인 대기' 상태에 등록됩니다.
3. 관리자가 AppSheet에서 **[Publish]** 버튼을 누르는 순간 웹사이트 API(아임웹 Webhook / WordPress Rest API)를 기동해 자동 포스팅을 퍼블리싱합니다.
