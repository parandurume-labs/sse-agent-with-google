# 🔍 Agent_Grant_Searcher.md (지원사업 타겟팅 필터링 룰)

이 문서는 지원사업 자동 스캐너(Module A)가 새로운 공고를 발견했을 때, 사회연대경제(SSE) 조직에 알맞은 공고인지 적합도 점수($S$)를 평가하는 심사 기준입니다.

---

## 🎯 조직 매칭 적합도 평가 점수 ($S \in [1, 10]$) 계산 기준

### 1. 가점 요인 (최대 +6점)
- **지원 대상 적합성 (최대 +3점)**:
  - 공고문 내에 '사회적기업', '예비사회적기업', '협동조합', '마을기업', '자활기업', '소셜벤처' 등 사회연대경제 관련 키워드가 명시된 경우 (+3점)
  - 일반 중소기업 대상이나 소상공인 대상이지만 SSE 조직도 신청 가능한 경우 (+1.5점)
- **사업 분야 일치성 (최대 +2점)**:
  - 디지털 트윈, WebXR, ICT 혁신, 지역 활성화, 취약계층 일자리 창출 분야 (+2점)
  - 일반 마케팅, 시제품 제작 지원, R&D 개발 자금 (+1점)
- **지역 적합성 (최대 +1점)**:
  - 관할 지역(인천, 송도 등) 또는 전국 단위 사업 (+1점)

### 2. 감점 및 제외 요인 (최대 -4점 / 즉시 탈락)
- **제한 업종 및 자격 요건**:
  - 대기업 전용 사업이거나 SSE 조직의 신청이 원천적으로 불가능한 경우 (-10점, 자동 제외)
  - 자부담(Matching Fund) 비율이 50% 이상으로 지나치게 높은 경우 (-2점)
